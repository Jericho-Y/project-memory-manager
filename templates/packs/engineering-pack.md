# Engineering Pack

Purpose: Optional engineering documents for architecture, APIs, database, and integrations.
Read when: Architecture, API, database, integration, or shared technical contracts are part of the task.
Skip when: The task is local and does not alter technical contracts.

Create only the files that contain real facts:

```text
docs/04-technical/technical-architecture.md
docs/04-technical/api-spec.md
docs/04-technical/database-design.md
docs/04-technical/integration-plan.md
```

Keep contracts concise and point to implementation files instead of copying code.
