Purpose: Optional design documents for IA, page map, UI rules, and content guidance.
Read when: UI, navigation, layout, interaction, or copy changes are part of the task.
Skip when: The task does not affect user experience.

Create only the files that contain real facts:

```text

docs/03-design/page-map.md
docs/03-design/ui-ux-guidelines.md

```
